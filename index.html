Le Concept du Jeu

Pierre-Feuille-Ciseaux est un jeu de main populaire où :

  • La Pierre bat les Ciseaux (les casse)
  • Les Ciseaux battent la Feuille (la coupent)
  • La Feuille bat la Pierre (l'enveloppe)

Dans cette version numérique, vous jouerez contre l'ordinateur qui fera des choix aléatoires.

Structure HTML de base

Créez la structure HTML qui contiendra notre jeu :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Pierre-Feuille-Ciseaux</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="game-container">
        <h1>Pierre-Feuille-Ciseaux</h1>
        <div class="choices">

    <script src="script.js"></script>
</body>
</html>

3

Logique du jeu en JavaScript

Implémentez la logique principale dans script.js :

// Variables pour stocker les scores
let scoreJoueur = 0;
let scoreOrdinateur = 0;

// Récupération des éléments du DOM
const pierreBtn = document.getElementById('pierre');
const feuilleBtn = document.getElementById('feuille');
const ciseauxBtn = document.getElementById('ciseaux');
const resultatDiv = document.getElementById('resultat');
const scoresDiv = document.getElementById('scores');

// Fonction pour le choix aléatoire de l'ordinateur
function choixOrdinateur() {
    const choix = ['pierre', 'feuille', 'ciseaux'];
    const randomIndex = Math.floor(Math.random() * 3);
    return choix[randomIndex];
}

// Fonction pour déterminer le gagnant
function determinerGagnant(choixJoueur, choixOrdinateur) {
    if (choixJoueur === choixOrdinateur) {
        return 'Égalité !';
    }

    if (
        (choixJoueur === 'pierre' && choixOrdinateur === 'ciseaux') ||
        (choixJoueur === 'feuille' && choixOrdinateur === 'pierre') ||
        (choixJoueur === 'ciseaux' && choixOrdinateur === 'feuille')
    ) {
        scoreJoueur++;
        return 'Vous gagnez !';
    } else {
        scoreOrdinateur++;
        return 'L\'ordinateur gagne !';
    }
}

// Gestion des clics sur les boutons
pierreBtn.addEventListener('click', () => jouer('pierre'));
feuilleBtn.addEventListener('click', () => jouer('feuille'));
ciseauxBtn.addEventListener('click', () => jouer('ciseaux'));

// Fonction principale du jeu
function jouer(choixJoueur) {
    const choixOrdi = choixOrdinateur();
    const resultat = determinerGagnant(choixJoueur, choixOrdi);
    
    resultatDiv.textContent = `Vous: ${choixJoueur} | Ordinateur: ${choixOrdi}. ${resultat}`;
    scoresDiv.textContent = `Score: Vous ${scoreJoueur} - ${scoreOrdinateur} Ordinateur`;
}
4

Améliorations possibles

Quelques idées pour améliorer votre jeu :

Animations

Ajoutez des animations CSS pour les choix et les résultats.

@keyframes shake {
    0%, 100% { transform: translateX(0); }
    25% { transform: translateX(-5px); }
    75% { transform: translateX(5px); }
}

.animate-shake {
    animation: shake 0.3s ease-in-out;
}
Effets sonores

Intégrez des sons pour chaque action.

function jouerSon(type) {
    const son = new Audio(`sounds/${type}.mp3`);
    son.play();
}

// Appeler dans la fonction jouer()
jouerSon(choixJoueur);
